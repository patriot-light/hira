# Hira Project Documentation

This `docs/` folder contains the full developer and operator documentation for the Hira project.

Files

- `INSTALLATION.md`: Setup and install steps for local development.
- `ARCHITECTURE.md`: High-level architecture and component responsibilities.
- `BACKEND.md`: Backend design, configuration, and run instructions.
- `FRONTEND.md`: Frontend design, scripts, and build/deploy steps.
- `API.md`: API overview and example endpoints.
- `DEVELOPMENT.md`: Developer workflows, common commands, and environment variables.
- `TESTING.md`: How to run tests and interpret results.
- `CONTRIBUTING.md`: Guidelines for contributing, coding standards, and PR process.
- `OPERATIONS.md`: Deployment and operational runbook.
- `DESIGN_GUIDELINES.md`: Design tokens and UX guidelines.

Start with `INSTALLATION.md` for quick local setup.
